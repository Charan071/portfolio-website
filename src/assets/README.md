# Assets Folder

## How to Add Your Images

### Profile Picture
1. Add your profile picture here as: `profile.jpg` or `profile.png`
2. The image will automatically appear in the hero section

### Project Images
Add your project images as:
- `project1.jpg` (or .png) - For your first project
- `project2.jpg` (or .png) - For your second project  
- `project3.jpg` (or .png) - For your third project

The images will automatically display in the Projects section.

### Recommended Image Sizes
- **Profile Picture**: 400x400px (square)
- **Project Images**: 800x600px (landscape) or any 4:3 ratio
